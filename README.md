# Inventory Agent

Status: Completed and verified.

---

## Overview

Inventory Agent is a Google Apps Script automation project designed to manage electronic inventory records through Google Sheets and chat-based input.

The system automates inventory updates and reduces manual spreadsheet handling.

---

## Features

- Inventory item management
- Data standardization
- Duplicate reduction
- Google Sheets automation
- Chat-based inventory updates
- Validation and update workflows

---

## Technologies Used

- Google Apps Script
- Google Sheets
- Google Chat
- JavaScript
